FeO Boy is an emulator for the Nintendo Game Boy.

Operation

Load a cartridge ROM by clicking the "Select a ROM" button above. If you don't have a ROM, check out these test ROMs written by Doug Lanford.

WebGPU

FeO Boy uses WebGPU to render. WebGPU is a highly experimental technology. It is unstable and may stop working at any time. To enable WebGPU in your browser, please read these instructions.

FeO Boy was last tested on Firefox Nightly 85.0.

Technical Details

FeO Boy is built using Rust, WASM, wgpu, and WebGPU. The source code is available on GitHub.
